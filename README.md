✅ Modern To-Do List Web App

This project is a modern, fully responsive To-Do List web application designed to improve productivity and simplify daily task management. Built with a focus on usability and clean design, the app provides an intuitive interface where users can organize and track their tasks efficiently.

The application supports secure user interaction, allowing users to manage their personal task lists with ease. Core functionalities include adding, editing, deleting, and marking tasks as completed. All data is stored persistently, ensuring tasks remain available even after refreshing or reopening the application.

A standout feature of this app is its integrated real-time clock with multi-time zone support. Users can view and switch between different regional times instantly, making it especially useful for those working across multiple locations or schedules.

To enhance personalization, the app includes customizable themes such as light and dark modes, allowing users to tailor the interface to their preference. Smooth animations, transitions, and interactive UI elements create a visually engaging and seamless user experience.

The design follows a clean, modern dashboard layout inspired by a reference UI, ensuring both aesthetics and functionality are balanced. The application is fully responsive and optimized for all devices, including mobile phones, tablets, laptops, and desktops.

Key Features:

- Add, edit, delete, and complete tasks
- Persistent data storage (local storage or backend integration)
- Real-time clock with multi-time zone support
- Light and dark mode theme customization
- Smooth animations and interactive UI
- Clean, modern dashboard-style layout
- Fully responsive across all screen sizes
